# Hub_Express_Distribution_Customer_Behavior
This project is an end-to-end data analysis case study focused on understanding customer purchasing behavior for Hub Express Distribution. The objective is to uncover actionable insights that can help the business improve sales, customer retention, and product performance through data-driven decision-making.
# Tools Used
Python
Jupyter
Pandas
SQL
Power BI
# Work Flow
1. Imported the dataset into Jupyter.
2. Cleaned and prepared the data using Pandas.
3. Loaded the cleaned data into PostgreSQL.
4. Performed SQL queries to answer business questions.
5. Created DAX measures and KPIs in Power BI.
6. Designed an interactive dashboard.
7. Generated insights and recommendations.
